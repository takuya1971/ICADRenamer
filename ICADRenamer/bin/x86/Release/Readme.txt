◇ 概要 ◇
  iCADのファイル名と図面番号変更作業を自動化します。
  コピー元フォルダとコピー先フォルダを指定して、M番、署名する名前を指定すると、コピー元の全ファイルのパーツ名・図面番号・日付・署名を変更します。
  オプションで改訂注記も削除できます。

◇ 動作条件 ◇
  Windows 10 64bit
  .NET Framework 4.8
  iCAD SX
  ICAD SDK+実行権

◇ ファイル構成 ◇

◇ インストール ◇
  Setup.exeをダブルクリックしてダイアログに従ってインストールしてください。
  .NET Framework 4.8は必要に応じてインストールされます。

◇ アンインストール ◇
  アプリの追加と削除からアンインストールしてください。

◇ つかいかた ◇
  1. メイン画面の4つのボックスへ入力します。コピー先フォルダ・コピー元フォルダ・M番・署名です。
  2. 実行ボタンを押すと、処理が始まります。
  3. モデルの規模によっては処理に時間がかかります。進捗画面を見ながら、気長に待ってください。
  4. 終了すると、コピー先フォルダに変換結果のCSVファイルが作成されます。
